if标签

where标签

foreach标签

诸如：select * from user id in（1,2,….） …..的语句

foreach的标签：

  1. collection：代表要遍历的集合元素，编写时不能写#{}
  2. open：代表语句开始的部分
  3. close：代表语句结束的部分
  4. separator：遍历元素之间的分隔符
  5. item：代表遍历集合的每个元素的变量名

定义代码片段

当重复的语句过多的时候，可以自定义代码片段

引用自定义的代码片段：include标签
